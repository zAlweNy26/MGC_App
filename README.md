# MG Clinic
App made for the Malta & Gozo Chiropratic clinics.

## Come avviarlo :
    - 'flutter run'
    - 'flutter build apk'
    - 'flutter build appbundle'

## Controllare aggiornamenti delle dipendenze :
    - 'flutter doctor'
    - 'flutter upgrade'
